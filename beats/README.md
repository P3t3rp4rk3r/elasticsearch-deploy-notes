# Beats
> Beats is a free and open platform for single-purpose data shippers. They send data from hundreds or thousands of machines and systems to Logstash or Elasticsearch.

_*Recommended to install beats as systemd unit instead of docker container._
* Community [beats list](https://www.elastic.co/guide/en/beats/libbeat/current/community-beats.html)
